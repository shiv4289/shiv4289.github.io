If you are into algorithms, you would realize that Primality testing is a very important topic. In naive terms Primality testing is finding if a number is prime or not. Lets discuss a few tricks for the same.

NAIVE ALGORITHM:


def primes_sieve2(limit):
    a = [True] * limit                          # Initialize the primality list
    a[0] = a[1] = False

    for (i, isprime) in enumerate(a):
        if isprime:
            yield i
            for n in xrange(i*i, limit, i):     # Mark factors non-prime
                a[n] = False
