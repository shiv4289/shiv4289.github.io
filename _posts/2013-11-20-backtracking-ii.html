In the previous post on backtracking, we saw some basic examples of backtracking. If you are not familiar with this technique I recommend reading the previous post first. Lets now look at some more codes. Most of this codes are inspired from from geeksforgeeks-a great repository of algorithms. The first code we see today prints all possible strings of length k given a set of n characters. Note that given a set of n characters, we can find k^n strings of length k comprising of characters from the given set.

[sourcecode language="cpp"]
#include <cstdio>
#include <iostream>
#include <string>
using namespace std;

void generate_all_len_k(string s, string p, int n, int k)
{
if(k==0){ //base case
cout<<p<<"\t";
return;
}
for(int i=0;i<n;i++){ //iterate over all characters in the given string
string newp=p+s[i]; //add this character at the end of the prefix string
generate_all_len_k(s,newp,n,k-1); //find all permutations of the other k-1 characters drawn from the n given characters
}
}

int main()
{
string s("AB");
string prefix(""); // we start with an empty prefix.
generate_all_len_k(s, prefix,2,3);
return 0;
}
